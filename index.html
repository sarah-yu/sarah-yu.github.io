Piece Peace

React.js Express.js Node.js MongoDB HTML/CSS

Live / Code

Piece Peace is an image management tool inspired by Pinterest. Users can save images that inspire them and organize them by categories of their choosing by creating "mood boards". The app currently features CRUD functionality, and I am working on a bookmarklet so that users can more easily pin images that they find around the web.

Murakami

Vue.js Express.js Node.js MongoDB HTML/CSS

Live / Code

A Haruki Murakami random quote generator, designed to evoke the same sense of simplicity and surrealness present throughout the author's books. I built this after a Teach Yourself Day in class, where I taught myself Vue.js. The Vue.js front-end is deployed to Surge, and the back-end is hosted on Heroku with mLab supporting the Mongo database.

Simon Learns Chinese

JavaScript/jQuery HTML/CSS

Live / Code

Simon Learns Chinese is based on the concept of the memory game, Simon, and inspired by gamified language learning apps. The purpose of the game is to test users' memories while allowing users to gain a basic but intuitive understanding of the Chinese language through repeated exposure to simple vocabulary and grammar patterns.
